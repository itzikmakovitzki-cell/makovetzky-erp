# איך מייצאים ל-Excel

מדריך כיס — איפה הכפתורים, מה מקבלים, איך פותחים.

:::info
**מי יכול?** ייצוא בסיסי — {role:admin} ו-{role:employee}. ייצוא של נתוני כסף — {role:admin} בלבד.
:::

## איפה הכפתור "ייצא לאקסל"?

יש כפתור בכל אחד מהמסכים האלה:

| מסך | מה מקבלים |
|---|---|
| `/permits` | רשימת היתרים פעילים — לקוח, פרויקט, רשות, התקדמות, סטטוס |
| `/permits/[id]` משימות | **דוח טופס 4 / תעודת גמר** — סטיילינג מלא, מקובץ לקטגוריות |
| `/tasks` | כל המשימות במערכת — לפי הסינון הנוכחי |
| `/my-tasks` | המשימות שלכם — לפי הסינון הנוכחי |
| `/clients` | רשימת לקוחות + פרטי קשר |
| `/projects` | רשימת פרויקטים |
| `/inbox` | מסמכים שנקלטו ב-WhatsApp |
| `/finances` {role:admin} | אבני דרך לחיוב + סטטוס תשלום |
| `/finances/supplier-commissions` {role:admin} | עמלות מספקים ברמת הקצאה |
| `/calendar` | תאריכי יעד מאוחדים לחודש |

## ייבוא מאקסל

לחלק מהמסכים יש גם **ייבוא**:

- `/clients` — ייבוא CSV/Excel של לקוחות
- `/permits/[id]` משימות — ייבוא של אותו פורמט שיוצא מהייצוא, להתאמות גורפות

המערכת מזהה את הפורמט וממפה אוטומטית. שגיאות מוצגות פר שורה.
