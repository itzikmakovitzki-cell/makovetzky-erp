# איך שולחים תזכורת WhatsApp לאחראי

מדריך כיס — 10 שניות, 2 לחיצות.

:::info
**מי יכול?** {role:admin} ו-{role:employee} — על כל משימה שמראה את האחראי.
**מה צריך מראש?** לאחראי יש מספר טלפון מוגדר ב-`/settings/users`.
:::

## השלבים

1. בכל מסך שמראה משימה (לדוגמה `/my-tasks`, `/tasks`, לשונית משימות של היתר) — מצאו את **הכפתור הירוק 📱** ליד שם האחראי.
2. לחיצה אחת עושה שני דברים בו-זמנית:
   - **מעתיקה ל-clipboard** הודעה מוכנה.
   - **פותחת `wa.me/[phone]?text=...`** בכרטיסייה חדשה.

ההודעה המוכנה:

```
היי [שם],
תזכורת לגבי משימה: [שם משימה]
בפרויקט [שם פרויקט].
אפשר סטטוס?
```

3. בכרטיסיית WhatsApp שנפתחה — לוחצים שלח. זהו.

## וריאציות

:::cols
**📱 משימה גם בנייד**
ב-`/my-tasks` במובייל — אותו כפתור 📱 בכרטיס. עובד עם WhatsApp האפליקציה.
---
**💼 שליחה ללקוח**
זה תרחיש אחר — דרך `/clients/[id]` בפאנל WhatsApp. ראו [WhatsApp ב-ERP](/guides/whatsapp).
:::

## אם משהו השתבש

:::warning
**אין מספר לאחראי?** הכפתור פתוח, ההודעה תועתק ל-clipboard, אבל wa.me לא ייפתח. הוסיפו טלפון ל-`/settings/users`.
:::

:::tip
**Green API מוגדר אצלכם?** לקוחות שמסומנים "ידני בלבד" יקבלו כפתור `📤 שלח עכשיו` בפרופיל — שליחה ישירה דרך השרת. ראו [WhatsApp ב-ERP](/guides/whatsapp).
:::
