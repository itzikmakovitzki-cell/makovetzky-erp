# Command Palette — חיפוש מהיר עולמי

מדריך כיס — איך מגיעים לכל דבר ב-3 הקשות.

:::info
**מי יכול?** כולם — {role:admin}, {role:employee}, {role:contractor}.
**מאיפה פותחים?** מכל מסך — <kbd>Ctrl+K</kbd> / <kbd>⌘K</kbd>, או כפתור 🔍 בסיידבר/במובייל.
:::

## איך זה נראה

:::frame title="Command Palette אחרי הקשת ⌘K"
🔍 `מגדל המ___`

**לקוחות**
👤 יעקב כהן בנייה — מגדל המשי

**פרויקטים**
📁 מגדל המשי

**היתרים**
📜 2026/417 — מגדל המשי — היתר ראשי
📜 2026/418 — מגדל המשי — שינויים פנימיים

⬆️⬇️ ניווט · ↵ פתח · ⎋ סגור
:::

## במה הוא מחפש?

- **שמות לקוחות** (`Client.companyName`)
- **שמות פרויקטים** (`MasterDeal.name`)
- **שמות היתרים** (`Permit.name`)
- **מספרי היתרים** (`Permit.permitNumber`)

חיפוש case-insensitive substring. תוצאות מקובצות לפי סוג עם hrefs ישירים.

## ניווט במקלדת

| מקש | פעולה |
|---|---|
| <kbd>↑</kbd> <kbd>↓</kbd> | מעלה/מטה בין תוצאות |
| <kbd>↵ Enter</kbd> | פתח תוצאה נבחרת |
| <kbd>Esc</kbd> | סגירה |
| <kbd>Ctrl+K</kbd> שוב | סגירה (toggle) |

## איך לרכוש הרגל

:::tip
**3 ימים ראשונים** — כל פעם שאתם מסיידברים, עצרו, פתחו ⌘K, חפשו, וסגרו. זה רק נוח יותר מהסיידבר אחרי קצת תרגול.
:::

:::tip
**זוכרים חלק מהשם** — מספיק "מגדל המ" כדי למצוא "מגדל המשי". אל תקלידו שמות מלאים.
:::

:::tip
**מספר היתר** — מהיר במיוחד. אם זוכרים `2026/417` — הקלידו את זה ועוברים ישר להיתר.
:::

## מה זה לא עושה (ומה הלאה)

:::warning
- **לא מחפש משימות** עדיין — לזה יש את `/tasks`.
- **לא מחפש מסמכים** — לזה יש את `/inbox` ולשונית מסמכים של ההיתר.
- **לא מחפש הערות** — לזה יש את הלשונית הערות.
:::

זה במכוון — Command Palette הוא **קיצור ניווט**, לא חיפוש כלל-מערכתי. הוא מהיר כי הוא מוגבל.
