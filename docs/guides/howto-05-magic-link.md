# איך משתמשים ב-Magic Link

מדריך כיס — איך נותנים לעובד שטח להעלות מסמך **בלי שיהיה לו משתמש במערכת**.

:::info
**מי יכול?** {role:admin} ו-{role:employee} — על כל משימה.
**איך זה עובד?** מייצרים קישור עם token ייחודי, שולחים בוואטסאפ. הקישור פותח דף ציבורי עם פרטי המשימה + טופס העלאה.
:::

- שייך לכם לעולמים

## השלבים

1. בלשונית **משימות** של ההיתר, מצאו את המשימה הרצויה.
2. לחצו על הכפתור **🔗** ליד שם המשימה.
3. בדיאלוג שייפתח — לחצו **"📋 העתק קישור"** או **"📱 שלח ב-WhatsApp"**.
4. שלחו לעובד השטח (אם בחרתם "שלח ב-WhatsApp" — wa.me נפתח עם הטקסט מוכן).

## מה העובד רואה כשהוא לוחץ?

דף ציבורי `/m/[token]` שמראה:

- שם המשימה והפרויקט
- תיאור המשימה
- כפתור "העלה קובץ"

הוא לא רואה את שאר המערכת — רק את המשימה הספציפית הזו.

## מה קורה כשהוא מעלה?

הקובץ נוחת כ-`Document` תחת ההיתר עם:

- `source: "MAGIC_LINK"`
- `uploadedById: null` (אנונימי)
- גרסה מחושבת אוטומטית
- מאשר ב-{role:admin} אם רוצים

## אם משהו השתבש

:::warning
**הקישור פג תוקף?** אחרי 48 שעות הוא לא יעבוד. צרו חדש מאותו דיאלוג.
:::

:::tip
**רוצים לבטל קישור פעיל?** הסירו אותו דרך המשימה (כפתור 🗑 בדיאלוג Magic Link). שימושי אם העובד עזב או טעיתם בשליחה.
:::

:::danger
**אל תשתפו Magic Link בערוץ פתוח** (קבוצת WhatsApp פתוחה, מייל לרשימת תפוצה). הקישור = אנונימי, מי שמקבל אותו יכול להעלות. בדקו את הנמען.
:::
