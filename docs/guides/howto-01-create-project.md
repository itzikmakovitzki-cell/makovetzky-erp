# איך יוצרים פרויקט חדש

מדריך כיס — 60 שניות, 6 שלבים.

:::info
**מי יכול?** {role:admin} ו-{role:employee}.
**מה צריך מראש?** רשות וסוג בניין מוגדרים (`/settings/authorities`, `/settings/building-types`). תבניות משימות לשילוב — מאוד מומלץ.
:::

## השלבים

1. כנסו ל-`/permits/new` (או מ-`/projects` → כפתור "+ פרויקט חדש").
2. **לקוח** — בחרו קיים מהרשימה, או "**+ לקוח חדש**" ומלאו שם + טלפון.
3. **פרטי פרויקט** — שם פרויקט + תאריך התחלה צפוי.
4. **פרטי היתר** — שם היתר, מספר היתר, רשות (dropdown), סוג בניין (dropdown).
5. **בניינים** — מספר בניינים + prefix (לדוגמה: "בניין"). ייווצרו אוטומטית בניין A, בניין B...
6. ודאו ש-**"צור משימות מתבניות" ✅ דלוקה** (ברירת מחדל) → [ שמור והמשך ].

## מה קורה אחר כך

:::success
המערכת תיצור במכה: לקוח (אם חדש) · פרויקט · היתר · בניינים · ואת כל המשימות מהתבניות המתאימות (רשות × סוג בניין) — כולל תלויות ותאריכי יעד מחושבים.
:::

תועברו לדף ההיתר עם 5 לשוניות. הלשונית הראשונה (משימות) תהיה כבר מלאה.

## אם משהו השתבש

:::warning
**רשימת המשימות ריקה?** סימן שאין תבניות לצירוף רשות+סוג בניין שבחרתם. עברו ל-`/settings/templates`, בחרו את הצירוף, והוסיפו תבניות. אחר כך — מחקו את ההיתר מ-`/permits` ויצרו מחדש.
:::

:::tip
**רוצים להוסיף עוד היתר לפרויקט קיים?** מ-`/projects/[id]` → "הוסף היתר" → דיאלוג בלי הזנה מחדש של הלקוח.
:::
