# התחלה מהירה

ברוכים הבאים ל-Makovetzky ERP — מערכת ניהול הבירוקרטיה והרישוי לקבלנים. המדריך הזה הוא נקודת הכניסה: אחרי שתעבור עליו תכיר את ההיררכיה, את 4 התפקידים, ואת ה-12 עמודי הניווט הראשיים.

## ההיררכיה — 4 רמות שמרכיבות כל פרויקט

```
לקוח (Client)
  └── פרויקט / עסקה (MasterDeal)
       └── היתר (Permit)              ← הלב של היומיום
            └── בניין (Building)
```

זה אילוץ קשה — אי אפשר ליצור היתר בלי פרויקט, ואי אפשר ליצור פרויקט בלי לקוח. בפועל זה מטופל בטופס יצירה אחד שיוצר את שלושת הרמות במכה אחת (`/permits/new`).

**מה תלוי באיזו רמה?**

| ישות | רמת אב |
|---|---|
| משימות (Tasks) | היתר |
| אבני דרך לחיוב (Billing Milestones) | היתר |
| מסמכים (Documents) | היתר (אפשר לתייג לבניין או למשימה) |
| הערות (Notes) | היתר |
| הגשות לרשות (Authority Submissions) | היתר + קטגוריה |
| אבני דרך של עסקה (Deal Milestones) | פרויקט |

## 4 התפקידים — מי רואה מה

### 🔑 ADMIN — מנהל

רואה הכל, משנה הכל. גישה ל-12 העמודים בסיידבר. **לבד יכול:** מחיקות, ניהול משתמשים, סיסמאות, ניהול לקוחות וספקים, ניהול אבני דרך כספיות, הגדרות, סל מיחזור, פעולות bulk, ייצוא/ייבוא Excel/CSV, אישור הגשות לרשות, יומן פעולות.

**לא יכול:** להשבית את עצמו, להוריד את עצמו מ-ADMIN.

### 👤 EMPLOYEE — עובד

ניהול שוטף של היתרים ומשימות, **בלי גישה לכספים ולקוחות**.

**יכול:** לעדכן משימות, להעלות מסמכים, ליצור הערות, ליצור Magic Links, לטפל בתיבת WhatsApp, ליצור היתרים ופרויקטים חדשים, /המשימות שלי.

**לא יכול:** לראות אבני דרך כספיות, לראות לקוחות, לראות ספקים, לראות הגדרות. בסיידבר — הפריטים האסורים אפורים עם אייקון מנעול 🔒.

### 🤝 CONTRACTOR — קבלן/גורם חיצוני

משתמש בפורטל ייעודי (`/portal`), לא בלוח הבקרה הפנימי.

**רואה:** רק לקוחות שיש לו `PortalAccess` אליהם, רק את ההיתרים והמשימות המוקצות לו.

**יכול:** להעלות מסמכים למשימות שלו, לעדכן סטטוס משימות שלו.

### 🔓 PUBLIC — אורח בלי לוגין

4 זרימות ציבוריות, כל אחת עם הגנה ייחודית:

1. `/login` — התחברות בלבד.
2. `/m/[token]` — Magic Link לעובד שטח (48 שעות תוקף, מרובה-שימוש עד פקיעה).
3. `/quote/[id]` — צפייה וחתימה דיגיטלית על הצעת מחיר.
4. `/forgot-password` — בקשת איפוס סיסמה. אם המספר רשום אצל המנהל, נשלחת הודעת WhatsApp עם קישור איפוס (1 שעה תוקף, חד-פעמי).

## הניווט — מה יש בכל עמוד

### במחשב — סיידבר ימני

| תפריט | מי רואה | מה יש שם |
|---|---|---|
| **מבט-על** (`/`) | ADMIN, EMPLOYEE | דשבורד עם ברכה, פעולות שצריך לעשות, "השבוע", "מוכן להגשה", פעילות אחרונה |
| **המשימות שלי** (`/my-tasks`) | ADMIN, EMPLOYEE | תיבת המשימות האישית — סינון, Kanban, עריכה בשורה |
| **הצעות מחיר** (`/proposals`) | ADMIN | טיוטות → שליחה → חתימה → המרה לפרויקט |
| **פרויקטים** (`/projects`) | ADMIN, EMPLOYEE | רשימת כל הפרויקטים, פתיחה לדף פרויקט עם כל ההיתרים שלו |
| **היתרים** (`/permits`) | ADMIN, EMPLOYEE | רשימת היתרים פעילים / ארכיון, פתיחה ל-5 לשוניות |
| **לקוחות** (`/clients`) | ADMIN | רשימת לקוחות, ייצוא/ייבוא CSV |
| **משימות** (`/tasks`) | ADMIN, EMPLOYEE | תצוגה גלובלית של כל המשימות, מסננים, פעולות bulk |
| **תיבת WhatsApp** (`/inbox`) | ADMIN, EMPLOYEE | מסמכים שנכנסו דרך WhatsApp ועדיין לא שויכו |
| **כספים** (`/finances`) | ADMIN | סקירת אבני דרך לכל ההיתרים, סטטוס תשלום |
| **עמלות מספקים** (`/finances/supplier-commissions`) | ADMIN | מה שמגיע לי — לפי תקופה |
| **ספקים** (`/suppliers`) | ADMIN | ניהול ספקים, הקצאות, עמלות |
| **הגדרות** (`/settings`) | ADMIN | משתמשים, רשויות, סוגי בניינים, תבניות, סל מיחזור, יומן פעולות |
| **מדריכים** (`/guides`) | ADMIN, EMPLOYEE | הקבצים שאתם קוראים עכשיו |

### במובייל — סרגל תחתון

ב-4 הסלוטים הראשיים בתחתית: **מבט-על · פרויקטים · היתרים · הצעות**. כל השאר תחת כפתור **"עוד"** שפותח גיליון מהתחתית עם רשימה מלאה של פריטי הניווט + תפריט משתמש + יציאה.

כל מסכי הרשימה הראשיים מציגים **כרטיסים במובייל ולא טבלאות צפופות** — לחיצה על כרטיס פותחת את ההיתר/פרויקט/לקוח/משימה.

## איך להתחיל

**משתמש חדש (מנהל):**

1. ב-`/settings/users` הוסף את העובדים שלך. סמן תפקיד נכון (`ADMIN` / `EMPLOYEE` / `CONTRACTOR`). הוסף טלפון — חיוני לתזכורות WhatsApp.
2. ב-`/settings/authorities` הוסף את הרשויות שאתה עובד מולן (עירייה, הוועדה המקומית, הג"א, וכו'). מלא שדה "מחוז" וצור עבור כל אחת רשומות Wiki עם דרישות התקן שלה.
3. ב-`/settings/building-types` הוסף סוגי בניינים שאתה עובד עליהם (וילה, מגדל מגורים, מסחרי, וכו').
4. ב-`/settings/templates` בחר רשות + סוג בניין → תיצור תבניות משימות שיופיעו אוטומטית בכל היתר חדש שאתה פותח לשילוב הזה.
5. ב-`/clients` הוסף לקוח ראשון. אפשר גם דרך ייבוא CSV.
6. ב-`/permits/new` צור היתר ראשון — אבל לפני זה, קרא את [המדריך לטופס 4 קצה לקצה](/guides/form-4-end-to-end).

**משתמש חדש (עובד):**

1. **המשימות שלי** (`/my-tasks`) — זה הבית שלך. סינון לפי "השבוע" + "פעיל" יראה לך מה צריך לסיים בקרוב.
2. **משימות** (`/tasks`) — תצוגה גלובלית של כל המשימות במערכת. שימושית כדי לראות את המצב הכללי.
3. **תיבת WhatsApp** (`/inbox`) — אם הגיע מסמך דרך WhatsApp שלא משויך להיתר, הוא יחכה שם. שייך אותו או דחה עם סיבה.

## טיפים שכל אחד צריך להכיר מהיום הראשון

- **Command Palette ⌘K / Ctrl+K** — חיפוש מהיר של לקוח / פרויקט / היתר. הקש מכל מקום.
- **Scratchpad** — פנקס טיוטה צף בפינה הימנית-תחתונה, שומר אוטומטית בדפדפן. שימושי לרשימות לא-רישמיות.
- **תיבת WhatsApp** — מסמכים שנשלחו לבוט שלך מ-WhatsApp נוחתים שם. מומלץ להיכנס פעם ביום ולעשות Triage.
- **Magic Link** — כפתור 🔗 בכל משימה. יוצר קישור ייחודי שמאפשר לעובד שטח להעלות מסמכים בלי משתמש בכלל. שולחים אותו בוואטסאפ.
- **Audit Log** — כל פעולה במערכת מתועדת. ב-`/settings/audit-log` (admin בלבד) אפשר לראות מי עשה מה ומתי, עם סינון מלא.
