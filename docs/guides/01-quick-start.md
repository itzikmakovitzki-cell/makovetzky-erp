# התחלה מהירה

ברוכים הבאים ל-**Makovetzky ERP** — מערכת ניהול הבירוקרטיה והרישוי לקבלנים.
המדריך הזה הוא נקודת הכניסה: אחרי 5 דקות תכירו את ההיררכיה, את 4 התפקידים, ואת תפריט הניווט.

:::info
**איך לקרוא מדריך:** רוב המסכים שתפגשו פה הם **מוקאפים נאמנים** של המסכים האמיתיים — כל מה שמופיע בתוך מסגרת עם שלוש נקודות אדום/צהוב/ירוק זה איך זה ייראה לכם בפועל.
:::

## 1. ההיררכיה — 4 רמות שמרכיבות כל פרויקט

כל פרויקט במערכת בנוי מ-4 רמות מקוננות:

:::frame title="מבנה ההיררכיה — בכל פרויקט"
**👤 לקוח** (Client)
  └── **📁 פרויקט / עסקה** (MasterDeal)
        └── **📋 היתר** (Permit) ← *הלב של היומיום*
              └── **🏢 בניין** (Building)
:::

זה אילוץ קשה — **אי אפשר** ליצור היתר בלי פרויקט, ו**אי אפשר** ליצור פרויקט בלי לקוח. בפועל הכל מטופל בטופס יצירה אחד שיוצר את שלושת הרמות במכה אחת (`/permits/new`).

### מה תלוי באיזו רמה?

| ישות | רמת אב | למה זה חשוב |
|---|---|---|
| משימות (Tasks) | היתר | המשימה לא קופצת לפרויקט אחר |
| אבני דרך לחיוב | היתר | חיוב פר היתר ולא פר פרויקט |
| מסמכים | היתר (אופציה לתייג בניין/משימה) | סינון מהיר |
| הערות | היתר | הקשר ברור |
| הגשות לרשות | היתר + קטגוריה | סנכרון מעמדים |
| אבני דרך של עסקה | פרויקט | חוצה היתרים |

:::tip
**כלל אצבע:** אם אתם מתלבטים איפה להוסיף משהו — כנראה זה שייך להיתר. ההיתר הוא הנכס המרכזי של היומיום.
:::

## 2. 4 התפקידים — מי רואה מה

### {role:admin} ADMIN — מנהל

רואה הכל, משנה הכל. גישה ל-12 העמודים בסיידבר.

**רק הוא יכול:** מחיקות · ניהול משתמשים וסיסמאות · ניהול לקוחות וספקים · אבני דרך כספיות · הגדרות · סל מיחזור · פעולות bulk · ייצוא/ייבוא Excel/CSV · אישור הגשות לרשות · יומן פעולות.

:::warning
מנהל **לא יכול** להשבית את עצמו או להוריד את עצמו מ-ADMIN. זה מנגנון הגנה מבעיטה-מהמערכת בטעות.
:::

### {role:employee} EMPLOYEE — עובד

ניהול שוטף של היתרים ומשימות, **בלי גישה לכספים ולקוחות**.

:::cols
**✅ יכול:**
- לעדכן משימות
- להעלות מסמכים
- ליצור הערות
- ליצור Magic Links
- לטפל בתיבת WhatsApp
- ליצור היתרים ופרויקטים חדשים
- לראות "המשימות שלי"
---
**🔒 לא יכול:**
- לראות אבני דרך כספיות
- לראות לקוחות
- לראות ספקים
- לראות הגדרות
- בסיידבר, הפריטים האסורים אפורים עם מנעול 🔒
:::

### {role:contractor} CONTRACTOR — קבלן/גורם חיצוני

משתמש בפורטל ייעודי (`/portal`), לא בלוח הבקרה הפנימי.

**רואה:** רק לקוחות שיש לו `PortalAccess` אליהם, רק היתרים ומשימות שמוקצות לו.
**יכול:** להעלות מסמכים למשימות שלו, לעדכן סטטוס משימות שלו.

### {role:public} PUBLIC — אורח בלי לוגין

4 זרימות ציבוריות, כל אחת עם הגנה ייחודית:

1. `/login` — התחברות בלבד.
2. `/m/[token]` — Magic Link לעובד שטח (48 שעות תוקף, מרובה-שימוש עד פקיעה).
3. `/quote/[id]` — צפייה וחתימה דיגיטלית על הצעת מחיר.
4. `/forgot-password` — בקשת איפוס סיסמה. אם המספר רשום, נשלחת הודעת WhatsApp עם קישור איפוס (1 שעה תוקף, חד-פעמי).

## 3. הניווט — מפת כל העמודים

### במחשב — סיידבר ימני קבוע

הסיידבר מקובץ ל-3 קבוצות אקורדיון. ה"פינים" של הקבוצות נשארים תמיד למעלה גם כשהקבוצה סגורה.

:::frame title="הסיידבר — מבט-על של 12 העמודים"
🏠 **מבט-על** — דשבורד ראשי
✅ **המשימות שלי** — תיבת ה-PM האישית

▾ **תפעול**
&nbsp;&nbsp;&nbsp;📋 הצעות מחיר
&nbsp;&nbsp;&nbsp;📁 פרויקטים
&nbsp;&nbsp;&nbsp;📜 היתרים
&nbsp;&nbsp;&nbsp;✓ משימות (גלובלי)
&nbsp;&nbsp;&nbsp;💬 תיבת WhatsApp

▾ **לקוחות וכסף** {role:admin}
&nbsp;&nbsp;&nbsp;👥 לקוחות
&nbsp;&nbsp;&nbsp;💰 כספים
&nbsp;&nbsp;&nbsp;🤝 עמלות מספקים
&nbsp;&nbsp;&nbsp;🏗️ ספקים

▾ **מערכת**
&nbsp;&nbsp;&nbsp;⚙️ הגדרות {role:admin}
&nbsp;&nbsp;&nbsp;📖 מדריכים
:::

### במובייל — סרגל תחתון + "עוד"

ב-4 הסלוטים הראשיים בתחתית: **מבט-על · פרויקטים · היתרים · הצעות**. כל השאר תחת כפתור **"עוד"** שפותח גיליון מהתחתית עם רשימה מלאה + תפריט משתמש + יציאה.

:::success
כל מסכי הרשימה הראשיים מציגים **כרטיסים במובייל ולא טבלאות צפופות**. לחיצה על כרטיס פותחת את ההיתר/פרויקט/לקוח/משימה.
:::

## 4. איך להתחיל — צ׳קליסט ל-3 דקות הראשונות

### אתם מנהל ({role:admin})?

1. **משתמשים** — ב-`/settings/users` הוסיפו את העובדים שלכם. סמנו תפקיד נכון (`ADMIN` / `EMPLOYEE` / `CONTRACTOR`). הוסיפו טלפון — חיוני לתזכורות WhatsApp.
2. **רשויות** — ב-`/settings/authorities` הוסיפו את הרשויות שאתם עובדים מולן (עירייה, ועדה מקומית, הג"א, וכו'). לכל אחת צרו רשומות Wiki עם דרישות הסטנדרט שלה.
3. **סוגי בניינים** — ב-`/settings/building-types` הוסיפו את הסוגים (וילה, מגדל מגורים, מסחרי, וכו').
4. **תבניות משימות** — ב-`/settings/templates` בחרו רשות + סוג בניין → צרו תבניות שיופיעו אוטומטית בכל היתר חדש לשילוב הזה.
5. **לקוח ראשון** — ב-`/clients` הוסיפו לקוח. אפשר גם דרך ייבוא CSV.
6. **היתר ראשון** — ב-`/permits/new` צרו היתר. לפני זה — קראו את [המדריך לטופס 4 קצה לקצה](/guides/form-4-end-to-end).

### אתם עובד ({role:employee})?

1. **המשימות שלי** (`/my-tasks`) — זה הבית שלכם. סינון "השבוע" + "פעיל" יראה לכם מה צריך לסיים בקרוב.
2. **משימות** (`/tasks`) — תצוגה גלובלית של כל המשימות. שימושית למצב הכללי.
3. **תיבת WhatsApp** (`/inbox`) — אם הגיע מסמך דרך WhatsApp שלא משויך, הוא יחכה שם. שייכו או דחו עם סיבה.

## 5. 5 קיצורי דרך שכל אחד צריך להכיר

:::frame title="קיצורי המקלדת והכלים הקבועים"
<kbd>Ctrl+K</kbd> <kbd>⌘K</kbd> — **Command Palette**: חיפוש מהיר של לקוח / פרויקט / היתר מכל מסך.

📝 **Scratchpad** — פנקס טיוטה צף בפינה הימנית-תחתונה, שומר אוטומטית בדפדפן.

💬 **תיבת WhatsApp** (`/inbox`) — מסמכים שנשלחו לבוט נוחתים שם. היכנסו פעם ביום לטריאז'.

🔗 **Magic Link** — כפתור בכל משימה. יוצר קישור ייחודי לעובד שטח להעלות מסמכים בלי משתמש.

🔍 **Audit Log** (`/settings/audit-log`) — כל פעולה במערכת מתועדת. סינון מלא — מי עשה מה ומתי.
:::

:::tip
חיפוש מהיר עם <kbd>Ctrl+K</kbd> הוא 80% מהניווט שלכם אחרי השבוע הראשון. תרגלו אותו עכשיו, לפני שאתם רגילים לסיידבר.
:::
